Mystic Cartography vs Unicode — the swap, visualized

The website currently renders zodiac glyphs as Unicode characters (♈, ♉…) in three components: ZodiacIcon (72px icon), daily/ZodiacWheel (SVG wheel), and /signs/[sign] (oversized background). This page mocks each surface with the OLD Unicode rendering on the left and the proposed NEW Mystic Cartography SVG swap on the right — using the actual website styling (same DM Sans + Cormorant Garamond fonts, same color palette, same scrim utilities, same measurements).
What to judge: at the icon scale (72px), does the SVG carry its weight vs the typographic Unicode glyph? On the wheel, do the 12 SVGs read as a cohesive family at small size? On the /signs/aries hero, does the oversized background illustration upgrade or compete with the editorial Cormorant italic headline?

Surface 1 · ZodiacIcon (72px)

components/ZodiacIcon.tsx
Used wherever the site shows a sign with optional selection state — navbar sun-sign chip when profile exists, daily page signature wheel, zodiac-signs index. Existing component has pulsing glow aura, orbit rings with orbiting dots, shimmer sweep, sparkle dots — all preserved. Only the glyph itself is swapped.
